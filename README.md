# MAPAS DE KARNAUGH

Este programa C imprime mapa de Karnaugh
para 2, 3 e 4 variáveis.

## COMPILAÇÃO, USO E LIMPEZA

```

$ make
$ ./mapas.out 2 0111
$ ./mapas.out 3 01110111
$ ./mapas.out 4 0111011101110111
$ make clean

```
